# Car Price Prediction — Machine Learning

## Overview
Machine learning project to predict car prices
using Linear Regression and Random Forest.
Trained on 205 cars with 26 features.

## Results
| Model | R² Score | MAE |
|-------|----------|-----|
| Linear Regression | 0.8411 | $ 2,127.47 |
| **Random Forest** | **0.9556** | **$1,323.27** |

Random Forest achieved **95.56% accuracy!**

## Key Findings
- Engine size is the most important feature
- Horsepower strongly affects price
- Car brand significantly impacts price
- Random Forest outperforms Linear Regression

## Tools Used
- Python 3.11
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Dataset
- Source: Kaggle — Car Price Prediction
- Records: 205 cars
- Features: 26 columns
- Target: Price ($5,118 → $45,400)
